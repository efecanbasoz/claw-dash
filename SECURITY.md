# Security Policy

## Supported Versions

Security fixes are applied to the latest default branch.

## Reporting a Vulnerability

Please report vulnerabilities privately via GitHub Security Advisories.

Do not disclose sensitive issues publicly before maintainers triage them.

## Operational Safety

- Keep `ENABLE_DANGEROUS_OPERATIONS=false` unless you explicitly need host operations.
- Run this dashboard only in trusted environments.
- Place the service behind private networking and access controls.
